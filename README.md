### Requirements

- Python 3.5+ <br>
- Windows, Linux, or MacOS X 10.9+. <br>
- On Linux, requires libsdl2 (2.0.5+) and libomp5 to run. <br>

### Installation of libraries

1. **$ sudo apt-get update** <br>
2. **$ sudo apt-get install gcc python-dev python3-dev libsdl2-dev libffi-dev libomp5** <br>
3. **$ sudo pip3 install tcod** <br>

Open a terminal window, and navigate to your Downloads directory: <br>
1. **$ git clone https://github.com/XenonLab-Studio/the-dungeon-of-sin.git** <br>
2. **$ cd the-dungeon-of-sin** <br>
3. **$ python3 run.py** <br>
